# Employee Management System

A modern and responsive Employee Management System built using **React.js**. This application allows users to efficiently manage employee records through an intuitive and user-friendly interface.

## Features

- Add new employees
- Edit employee details
- Delete employee records
- Search and filter employees
- Responsive UI
- Component-based architecture using React

## Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3

## Purpose

This project was built to strengthen my understanding of React concepts such as components, props, state management, event handling, forms, and CRUD operations while creating a real-world application.
